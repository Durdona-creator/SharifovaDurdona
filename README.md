## 📌 Overview

FitZone Gym is a multi-page fitness website built using HTML, CSS, and JavaScript.  
It provides users with a simple and modern platform to explore gym memberships, register, and track their fitness activity.

---

## 🌐 Live Website
https://durdona-creator.github.io

---

## ✨ Features
- Multi-page navigation
- Responsive design
- Membership system
- Registration form
- Community activity records
- Personal dashboard
- Dynamic data display using JavaScript
